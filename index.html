HTML Блочні та вбудовані елементи



Блочні елементи


Блочний елемент завжди починається з нового рядка і займає всю доступну ширину (розтягується вліво та вправо, наскільки це можливо).

    Це блочні елементи в HTML:
  • header
    • main
  • div
    • p
  • h1-h6
    • ul
  • ol
    • li
  • footer

Елемент div

Елемент div часто використовується як контейнер для інших елементів HTML.

При використанні разом із CSS елемент div може використовуватися для стилювання блоків вмісту



Вбудовані елементи (рядкові)


Вбудований елемент не починається з нового рядка, і він займає лише стільки ширини, скільки потрібно.

    Це вбудовані елементи в HTML:
  • a
    • img
  • span
    • br
  • mark
Примітка:
Вбудований (рядковий) елемент не може містити в собі блочний елемент!
Елемент snap

Елемент span - це вбудований контейнер, який використовується для розмітки частини тексту або частини документа.

При використанні разом із CSS елемент span можна використовувати для стилізації частин тексту

    Підсумок
  1. Є два значення відображення: блочне та вбудоване
  2. Блочний елемент завжди починається з нового рядка і займає всю доступну ширину
  3. Вбудований елемент не починається з нового рядка, і він займає лише стільки ширини, скільки потрібно
  4. Елемент div є блочним і часто використовується як контейнер для інших елементів HTML
  5. Елемент span - це вбудований контейнер, який використовується для розмітки частини тексту або частини документа
